Strengths

  • Java is easy to learn. The language was designed in such a way that the syntax makes it easy for a new user to learn quickly.
  • Object Oriented. This allows the user to create reusable code, while also being much easier to read and work with.
  • Java does not rely on any one platform. It can run on any machine.

Weaknesses

  • Overloading.
  • No unsigned integer types.
  • Slow security patches from Oracle.

Originally created by Sun Microsystems, Java is now owned and maintained by Oracle. It is a very popular language among University campuses. This is because of the language's ease of use and its ability to introduce students to object-oriented programming.
